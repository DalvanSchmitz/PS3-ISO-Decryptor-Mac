<a name="english"></a>
## 🇬🇧 English

**PS3 ISO Decryptor** is a modern, elegant, and professional Java-based GUI for decrypting PlayStation 3 ISO files using DKEYs. Optimized for **macOS**, it provides a seamless "Zero-Config" experience for the preservation community.

### ✨ Key Features
- **Zero-Config Portability**: The `ps3dec` binary is bundled inside the JAR. No need to download extra dependencies or set PATHs.
- **Smart Game ID Discovery**: Automatically extracts the Title ID from the filename or ISO metadata.
- **One-Click DKEY Search**: Instant online lookup for the required decryption key based on the Game ID.
- **Batch Processing**: Decrypt your entire library at once with the powerful batch tab.
- **Native macOS Feel**: Dark mode UI, professional icons, and native system notifications.
- **Multi-language Support**: Fully localized in English and Portuguese.

### 🚀 Quick Start
1. **[Download the latest PS3-ISO-Decryptor.jar](https://github.com/DalvanSchmitz/PS3-ISO-Decryptor-Mac/raw/master/release/PS3-ISO-Decryptor.jar)**.
2. Ensure you have **Java 11+** installed.
3. Simply double-click the `.jar` to run!

> [!TIP]
> **macOS Security Note (Gatekeeper):**
> If you see a warning about "unverified developer", go to **System Settings > Privacy & Security** and click **"Open Anyway"** at the bottom.

## 🇧🇷 Português

O **PS3 ISO Decryptor** é uma interface gráfica (GUI) moderna e profissional escrita em Java para descriptografar arquivos ISO de PlayStation 3 usando chaves DKEY. Focado no **macOS**, ele oferece uma experiência "Zero-Config" para a comunidade de preservação de jogos.

### ✨ Funcionalidades
- **Portabilidade "Zero-Config"**: O binário `ps3dec` já vem embutido dentro do JAR. Você não precisa baixar dependências extras.
- **Descoberta Inteligente de Game ID**: Identifica o ID do jogo automaticamente pelo nome do arquivo ou metadados da ISO.
- **Busca de DKEY em Um Clique**: Consulta online instantânea para encontrar a chave de descriptografia necessária.
- **Processamento em Lote**: Descriptografe sua biblioteca inteira de uma vez só usando a aba de lote.
- **Visual Nativo macOS**: Interface estilo Dark Mode, ícones profissionais e notificações de sistema nativas.
- **Multi-idioma**: Traduzido 100% para Português e Inglês.

### 🚀 Início Rápido
1. **[Baixe a versão mais recente do PS3-ISO-Decryptor.jar](https://github.com/DalvanSchmitz/PS3-ISO-Decryptor-Mac/raw/master/release/PS3-ISO-Decryptor.jar)**.
2. Certifique-se de ter o **Java 11 ou superior** instalado.
3. Basta dar dois cliques no arquivo `.jar` para começar!

> [!TIP]
> **Nota para usuários de Mac (Gatekeeper):**
> Se aparecer um aviso de "desenvolvedor não verificado", vá em **Ajustes do Sistema > Privacidade e Segurança** e clique em **"Abrir Mesmo Assim"** no final da página.
